How to Install XLingPaper

Go to https://software.sil.org/xlingpaper/download/ to download and install XLingPaper and XMLMind Editor at the same time.
Note: this download is for the personal edition of XMLMind. If you do not qualify, follow the istructions here.

In order to publish to PDF, you must have a PDF viewer installed. I have found that Adobe Acrobat Reader works well.
However, make sure to uncheck the "optional offer" as it is bloatware
